<h1>Space Shooter</h1>
<h2>Jogo desenvolvido como atividade avaliativa em Programação Web</h2>
<h2>Como jogar</h1>
<ul>
  <li>Use as setas para mover a nave</li>
  <li>CLique espaço para atirar</li>
  <li>Ao ser atingido quatro vezes o jogo termina</li>
</ul>
